## About
There are expected to be five groups of particles that can **reasonably interact with each other and the environment** according to rules from the real world
- Solid
    - Dirt
    - Stone
    - Wood
    - Ice
- Powder
    - Sand
    - Snow
    - GunPowder
- Liquid
    - Water
    - Oil
    - Acid
    - Lava
- Spread
    - Fire
- Gas
    - Smoke
    - Steam
